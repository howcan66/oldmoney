SETUP STEPS
-----------

1. CLONE REPOSITORY
   cd C:\Git
   git clone https://github.com/howcan66/oldmoney

2. NAVIGATE TO PROJECT
   cd oldmoney

3. ADD FILES TO GIT
   git add .

4. COMMIT CHANGES
   git commit -m "Add website files with modern responsive design"

5. PUSH TO GITHUB
   git push origin main

6. ENABLE GITHUB PAGES
   - Go to https://github.com/howcan66/oldmoney
   - Click "Settings" tab
   - Click "Pages" in left sidebar
   - Under "Source": Select "main" branch
   - Select "/ (root)" folder
   - Click "Save"
   - Wait 1-2 minutes for deployment

7. ACCESS YOUR SITE
   Main links page: https://howcan66.github.io/oldmoney/oldmoneylinks.html
   Listing page: https://howcan66.github.io/oldmoney/listing.html

DESIGN FEATURES
---------------
- Responsive font sizing with clamp()
- Grid layouts for mobile, tablet, desktop
- Touch-optimized for mobile devices
- Retina display support
- Smooth animations and hover effects
- Shared CSS for consistent styling across all pages

MOBILE BREAKPOINTS
------------------
- Mobile: < 767px
- Tablet: 768px - 1024px
- Desktop: > 1024px

CREATING NEW PAGES
------------------
To create a new page with the same design:

1. Create new HTML file (e.g., newpage.html)
2. Add this in the <head> section:
   <link rel="stylesheet" href="styles.css">
3. Use these HTML structure classes:
   - .container (main wrapper)
   - .links-section or section (content sections)
   - .listing-card (for card layouts)
   - footer (bottom section)

UPDATING STYLES
---------------
To change the design for ALL pages:
- Edit styles.css
- All pages will automatically update
