Target Blank Demo

This page demonstrates how target="_blank" works. This page opened in a new tab/window because the link used target="_blank" attribute.

How This Works:

  • target="_blank" - Opens link in new tab/window
  • rel="noopener noreferrer" - Security protection
  • Prevents window.opener exploit
  • Use for external links and workflow interruptions

Example Code:

<a href="open_blank_demo.html" 
   target="_blank" 
   rel="noopener noreferrer">
   Open Blank Demo
</a>

You can close this tab/window to return to the previous page.
